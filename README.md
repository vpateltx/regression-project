# YouTube Watch Time Regression Model

This project uses linear regression to predict YouTube watch time based on video duration.

## Dataset

The dataset contains YouTube recommendation and engagement data, including:

- video duration
- watch time
- likes
- comments
- subscriptions
- category
- device type

## Model

The model uses:

- Linear Regression
- Train/test split
- R² score
- Mean Absolute Error

```bash
pip install pandas scikit-learn
python3 main.py
